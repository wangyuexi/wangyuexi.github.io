顺序表综合应用题

第一题

从顺序表中删除具有最小值的元素（假设唯一）并由函数返回被删元素。空出的位置由最后一个元素填补，若顺序表为空，则显示出错信息并退出运行。

bool Del_Min(SqList &L, ElemType &value){
    //删除顺序表L中最小元素的结点，并通过引用型参数value返回值
    //若成功则返回true，反之false
    if (L.length == 0)
        return false;		//表空退出
    value = L.data[0];
    int pos = 0;			//假设下标为0的元素值最小
    for (int i = 0;i < L.length;i++){//循环找最小值
        if (L.data[i] < value)	//value记录当前最小的值
            value = L.data[i];
        	pos = i;
    }
    L.data[pos] = L.data[L.length - 1];	
    L.length--;
    return value;
}

第二题

设计一个高效算法，将顺序表L的所有元素逆置，并要求算法的空间复杂度为O(1)。

void Reverse(SqList &L){
    ElemType e;
    for (int i = 0;i < L.length / 2;i++){
        e = L.data[i];
        L.data[i] = L.data[L.length - 1];
        L.data[L.length - 1] = e;
    }
}

第三题

对长度为n的顺序表L，编写一个时间复杂度为O(n)、空间复杂度为O(1)的算法，该算法删除线性表中所有值为x的数据元素。

void Del_x_l(SqList &L,ElemType x){
    int k = 0;
    for (int i = 0;i < L.length;i++){
        if (L.data[i] != x)
            L.data[k] = L.data[i];
        	k++;
    }
    L.length = k;
}

第四题

从有序顺序表中删除其值在给定值s与t之间（要求s<t）的所有元素,若s或t不合理或顺序表为空，则显示出错信息并退出运行。

bool Del_number(SqList &L,ElemType s,ElemType t){
    int i,j;
    if (L.length == 0 || s >= t)
        return false;
    for (i = 0;i < L.length && L.data[i] < s;i++);	//寻找值大于或等于s的第一个元素
    if (i > = L.length)
        return false;
    for (j = 0;j < L.length && L.data[j] <= t;j++);	//寻找值大于t的第一个元素
    for (;j < L.length;j++,i++){
        L.data[i] = L.data[j];						//前移，填补被删除的元素
    }
    L.length = i;
    return true;
}

第五题

从顺序表中删除其值在给定值s与t之间（包含s和t，要求s<t）的所有元素，若s或t不合理或顺序表为空，则显示出错信息并退出运行

bool Del_number(SqList &L,ElemType s,ElemType t){
    int i,k = 0;
    if (L.length == 0 || s >= t)
        return false;
    for (i = 0;i < L.length;i++){
        if (L.data[i] >= s && L.data[i] <= t)
            k++;
        else
            L.data[i - k] = L.data[i];			//当前元素前移k个位置
    }
    L.length -= k;
    return true;
}

bool Del_Same(SqList &L){
    if (L.length == 0)
        return false;
    for (int i = 0;int j = 1;j < = L.length;j++){
        if (L.data[i] != L.data[j])		//查找下一个与上一个元素不相同的元素
            L.data[++i] = L.data[j];	//找到后元素前移
    }
    L.length = i + 1;
    return true;
}

第七题

将两个有序顺序表合并为一个新的有序顺序表，并由函数返回结果顺序表。

bool Merge(SqList A,SqList B,SqList &C){
    if (A.length + B.length > C.length)
        return false;
    int i = 0,j = 0,k = 0;
    while (i < A.length && j < B.length)}{		//循环两两比较，小的存入结果集
    	if (A.data[i] <= B.data[j])
            C.data[k++] = A.data[i++];
    	else
            C.data[k++] = B.data[j++];
	}
	while (i < A.length){			//还剩下一个没有比较完的
        C.data[k++] = A.data[i++];
    }
	while (j < B.length){
        C.data[k++] = B.data[j++];
    }
	C.length = k;
	return true;
}

链表综合应用题

第一题

设计一个递归算法，删除不带头结点的单链表L中所有值为x的结点

void Del_X(LinkList &L,ElemType x){
    LNode *p;			//p指向待删除的结点
    if (L == NULL)		//递归出口
        return;
    if (L -> data == x) //若L所指结点的值为x
        p = L;			//删除*L，并让L指向下一个结点
    	L = L -> next;
    	free(p);
    	Del_x(L,x);		//递归调用
    else
        Del_x(L -> next,x) //递归调用
}

第二题

在带头结点的单链表L中，删除所有值为x的结点，并释放其空间，假设值为x的结点不唯一，编写算法实现以上操作。

void Del_x(LinkList &L,ElemType x){
    LNode *p = L -> next, *pre = L, *q;		//置p和pre的初始值
    while (p != NULL){
        if (p -> data == x)
            q = p;			  //q指向该结点
        	p = p -> next;
        	pre -> next = p;  //删除*q结点
        	free(q);
        else
            pre = p;
        	p = p -> next;
    }
}
