队列的顺序存储结构

队列的定义

//初始化操作

#define MaxSize 10			//定义队列中元素的最大个数
typedef struct{
    ElemType data[MaxSize];	//静态数组存放栈中元素
    int front,rear;			//队头指针和对尾指针
}SqQueue;

//初始化队列
void InitQueue(SqQueue &Q){
    Q.rear = Q.front = 0;	//初始时队头，队尾指向0
}

//判断队列是否为空
bool QueueEmpty(SqQueue Q){
    if (Q.rear == Q.front)
        return true;
    else
        return false;
}

入队操作(循环队列)

//入队
bool EnQueue(SqQueue &Q,ElemType x){
    if ((Q.rear + 1) % MaxSize == Q.front)
		return false;
    Q.data[Q.rear] = x;		//新元素入队尾
    Q.rear = (Q.rear + 1) % MaxSize;
}

出队操作(循环队列)

//出队(删除一个队头元素，并用x返回)
bool DeQueue(SqQueue &Q,ElemType &x){
    if (Q.rear == Q.front)
		return false;
    x = Q.data[Q.front];		//新元素入队尾
    Q.front = (Q.front + 1) % MaxSize;	//队头指针后移
    return true;
}

//获得队头元素的值，用x返回
bool GetHead(SqQueue Q,ElemType &x){
    if (Q.rear == Q.front)
		return false;
    x = Q.data[Q.front];		//新元素入队尾
    return true;
}

循环队列的判满

第一种方式

队头指针在队尾指针的下一个位置

  • 队列为空：Q.front = Q.rear
  • 队空已满：(Q.rear + 1 ) % MaxSize == Q.front
第二种方式

添加一个size显示当前队列长度

插入元素为size++

删除元素为size–

  • 队列为空：Q.size == 0
  • 队列已满：Q.size = = MaxSize
第三种方式

添加一个tag-增加元素值为1，删除元素值为0

若Q.front == Q.rear且tag == 1，则队列已满

若Q.front == Q.rear且tag == 0，则队列为空
